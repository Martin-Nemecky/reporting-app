# Getting Started

```bash
$ npm install
```

## Run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [https://localhost:3000](https://localhost:3000) with your browser to see the result.

## Used Technologies

- Next.js
- Material UI
- React
- TailwindCSS
- TypeScript
